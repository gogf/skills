# 重要规范

- 在使用GoFrame作为主要框架开发完整工程类型的项目时，应该首先需要安装GoFrame CLI开发工具。
- GoFrame工程规范中由开发工具自动维护的代码文件，不允许手动生成或修改，应当使用GoFrame CLI工具进行相关操作。
